# ⚡ Z-TOOLS | The Student OS
A high-performance, mobile-first utility suite designed to eliminate academic friction. Built with a focus on speed, precision, and professional document generation.

## 🚀 Live Demo
[View Z-Tools Live](https://swayamweb.github.io/z-tools/)

---

## 🛠️ Key Modules

### 📊 Grade Analytics
* **Semester Mode (SGPA):** Calculate current semester results based on credits and grades.
* **Cumulative Mode (CGPA):** Average previous semesters to track overall degree standing.
* **Smart Conversion:** Instant 10-point CGPA to Percentage conversion (using the 9.5 multiplier).

### 📄 Smart PDF Engine
* **Hybrid Export:** Combine typed notes and handwritten image attachments into a single, professional PDF.
* **Auto-Formatting:** Generates a structured document with a dedicated cover page and image pages.

### ✍️ Synopsis Pro
* **Abstract Generator:** Converts project titles and tech stacks into professional academic summaries.
* **Use Case:** Perfect for drafting project proposals and research paper introductions.

### 🕒 Deadline Pulse
* **Assignment Tracker:** A lightweight deadline manager with real-time "Days Left" countdowns to ensure zero missed submissions.

## 💻 Tech Stack
* **Frontend:** Tailwind CSS (UI/UX Architecture)
* **Fonts:** Plus Jakarta Sans
* **Logic:** Vanilla JavaScript (ES6+)
* **PDF Core:** jsPDF Library

## 📱 Installation (PWA Style)
To use Z-Tools as a native app:
1. Open the link in Chrome/Safari.
2. Tap **"Add to Home Screen"**.
3. Access Z-Tools instantly from your app drawer.
